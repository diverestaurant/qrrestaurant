# Architecture Decision Records

ADRs capture durable decisions that affect architecture, security, money, state, contracts or operations.

Template:

```text
# ADR-NNN — Title

Status:
Date:

## Decision
## Context
## Options considered
## Chosen approach
## Reason
## Tradeoffs
## Consequences and required tests
## Revisit triggers
```

Do not rewrite accepted ADR history silently. Supersede it with a new ADR and link both.
